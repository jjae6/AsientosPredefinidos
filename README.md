# AsientosPredefinidos
El plugin Asientos Predefinidos te permite agilizar tu contabilidad automatizando la creación de asientos
contables que utilizas de forma recurrente. En lugar de introducir manualmente cada cuenta y concepto,
puedes configurar plantillas que se rellenan automáticamente, ahorrándote tiempo y evitando errores.
- https://facturascripts.com/plugins/asientospredefinidos
